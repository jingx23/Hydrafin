<p align="center">
  <b>Hydrafin</b> is a personal fork of <a href="https://github.com/jellyfin/Swiftfin">Swiftfin</a> — a native iOS/tvOS client for the <a href="https://github.com/jellyfin/jellyfin">Jellyfin</a> media server. This fork replaces VLC with <b>MPV</b> for video playback and is intended as a base for future Xtream integration.
</p>

## 📖 Documentation

Upstream Swiftfin provides detailed documentation on the app and its development approach:

- [🎞️ Library Support](Documentation/libraries.md) — Information on **library compatibility** and supported media types.
